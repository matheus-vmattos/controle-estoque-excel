# Controle de Estoque Excel

> Sistema de controle de materiais com alertas automaticos, dashboards gerenciais e relatorios de movimentacao - construido em Excel com automacao via formulas avancadas.

---

## O que faz

Solucao completa para gestao de estoque em ambientes administrativos, desenvolvida para eliminar controle manual em planilhas desorganizadas. O sistema centraliza entradas, saidas e saldo de materiais com alertas visuais e dashboards de facil leitura.

**Funcionalidades principais:**
- Controle de entrada e saida de materiais
- Alertas automaticos de estoque minimo
- Dashboard gerencial com indicadores visuais
- Relatorio de movimentacao por periodo
- Categorizacao por tipo de material
- Historico completo de operacoes

---

## Stack

| Componente | Tecnologia |
|-----------|----------|
| Plataforma | Microsoft Excel |
| Automacao | Formulas avancadas (PROCV, SE, CONT.SE, SOMASES) |
| Visualizacao | Graficos dinamicos e formatacao condicional |
| Alertas | Validacao de dados + formatacao condicional |

---

## Por que este projeto

Desenvolvido a partir de uma necessidade real em ambiente corporativo: substituir controles manuais sujeitos a erros por um sistema automatizado que qualquer colaborador consiga operar sem treinamento tecnico.
